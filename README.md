# 4コマJSONラッシュ

SNSで使える4コマ漫画JSONメーカーをテーマにした、30秒スコアアタックのThree.jsミニゲームです。

## 目的

プレイヤーは光る「起承転結JSON」を集め、赤い崩壊プロンプトを避けます。スコアに応じて、画面内に4コマ漫画用のJSONサンプルとpromptサンプルが表示されます。

## 使い方

ローカルでは `index.html` をブラウザで開くだけで動きます。GitHub Pagesにもそのまま配置できます。バックエンド、DB、ログイン、npmビルドは不要です。

## 生成ファイル

- `index.html`
- `style.css`
- `app.js`
- `README.md`
