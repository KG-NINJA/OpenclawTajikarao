# 可視性チェック：遺跡バックグラウンドを守れ（Three.js / 30秒）

生成された背景画像が **CSSで見えているのにゲーム中に隠れてしまう** 問題を、30秒で目視検査できるスコアアタックです。

## 遊び方（目的）
- 30秒間でスコアを稼ぐ
- **光のキューブ**（良いオブジェクト）を集めるとスコア+＆可視性UP
- **赤いノイズ**（危険）に当たるとスコア-＆可視性DOWN（背景が暗く見えやすくなる＝隠蔽が進む）

背景（チチェン・イッツァ風の明るい遺跡）は **`#gameRoot` 内のCSSレイヤー** として常に表示され、Three.jsキャンバスは透過（alpha）で上に重なります。

## 操作
- PC：←→ / A D
- スマホ：画面上を左右にスワイプ（指の位置に追従）

## X共有
ゲーム内の「Xで共有」から、スコアと `#KGNINJA` を含む投稿ができます。
