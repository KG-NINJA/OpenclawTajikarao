# Operational Rules

This repository is maintained as a living system by an autonomous OpenClaw workflow.

## Mandatory rules

1. Every meaningful action must be logged in `AUTONOMOUS_AGENT_LOG.md`.
2. The activity log must include:
   - timestamp
   - task executed
   - result
   - next planned task
3. Commits must reference the corresponding log entry.
4. The autonomous workflow should prefer small, inspectable, reversible improvements.
5. Documentation, maintenance notes, and operational traces are first-class outputs of the system.
