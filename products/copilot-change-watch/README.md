# Copilot Change Watch

A tiny single-file HTML tool that turns messy GitHub / Copilot feature observations into a short change brief.

## Problem
Teams often notice GitHub and Copilot feature changes too late. Model availability, plan differences, and UI changes can create confusion and support overhead.

## Solution
Paste raw observations by date, plan, or screenshot note, then generate a concise change brief.

The tool includes a Tweet button so users can share interesting feature changes and the generated brief.
